CSS Sprite Image Demonstration

This project demonstrates the power of CSS sprites for optimizing web performance by combining multiple images into a single file.

Welcome to notSurf

This application showcases interactive navigation elements that use a single sprite sheet for all icons and their various states (normal, hover, and active).

You can navigate through the different sections using the bottom navigation bar. Each icon you see is a demonstration of precise background positioning on a sprite sheet.

The sprite technique helps in reducing the number of HTTP requests, which was a critical optimization for web performance in the era this project was originally built.

Benefits of CSS Sprites

  • Reduced HTTP Requests: One request for many images.
  • Faster Page Load: Minimal overhead for metadata.
  • Instant State Changes: No flickering when hovering.
  • Efficiency: Better compression for a single large image.
